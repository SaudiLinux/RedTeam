# 🔴 Red Team Interactive Framework

Professional Red Team Simulation & Web Assessment Toolkit  
Version: 3.0  
Platform: Linux / Python 3.x  

---

## 📌 Overview

Red Team Interactive Framework is a modular security assessment tool designed for:

- Authorized penetration testing  
- Red Team simulation  
- Web application exposure analysis  
- Automated reconnaissance  
- Risk scoring & graph-based exposure mapping  

It includes:

- Interactive MSF-style console  
- Multi-target concurrent scanning  
- Embedded local system shell  
- Automated recon chain
- CVE Lookup (NVD + Exploit-DB)
- Risk scoring based on CVSS
- Exposure graph export (JSON)
- Professional session logging  
- Caching of CVE results for faster execution
